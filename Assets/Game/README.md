읽어 보세요
===========
용도별로 폴더를 구분해 놓고 번호를 붙여서 정렬해 놓았습니다.

## 폴더 규칙
ABB. 폴더 이름
A: 대분류
B: 소분류

하위 폴더내에서는 추가적인 규칙을 만들어 폴더를 정리해주시기 바랍니다.

## 주의 사항
폴더 이름이나 순서가 마음에 들지 않는다고 다른 팀원들에게 통보하지 않고 수정하면
병합이 불가능해져서 팀 전체의 워크플로우가 망가질 수도 있습니다. 수정할 때는 반드시
팀원들과 협의하에 수정하도록 합시다.

빈 폴더는 가급적 만들지 말고 혹여나 필요하다면 Empty Folder라는 빈 파일 하나를
생성해주세요. Git에서 빈 폴더를 감지하지 못합니다.

파일 이름은 한글, 띄어쓰기 등이 들어가도 상관은 없지만 일정한 규칙을 사용해주세요.
